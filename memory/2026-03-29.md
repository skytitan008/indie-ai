### 2. Agent 谈判协议 🤝

**创建文件:**
- `src/multi_agent/negotiation.py` - 谈判协议核心（13.7KB）
- `demo/Agent 谈判协议演示.py` - 完整演示（6.3KB）
- `docs/Agent 谈判协议.md` - 详细文档（5.5KB）

**核心类:**
- `MessageType` - 9 种消息类型
- `NegotiationAgent` - 谈判 Agent
- `ContractNetProtocol` - 合同网协调器

**消息类型:**
- `TASK_ANNOUNCE` - 任务公告
- `BID` - 投标
- `AWARD` - 中标
- `ACCEPT/REJECT` - 接受/拒绝
- `PROPOSE/COUNTER_PROPOSE` - 提议/反提议
- `AGREE/REFUSE` - 同意/拒绝

**投标算法:**
```
投标值 = 基础成本 × (1 - 能力匹配度 × 0.3) × 负载系数 × 优先级系数
```

**演示场景:**
1. 基础任务分配 - 3 个 Agent 竞标
2. 多轮协商 - 提议与反提议
3. AIGC 视频工作流 - 导演→编剧→分镜→视频
4. 不同谈判策略 - 激进/保守/协作

**实验结果:**
```

💰 第 2 轮：投标

   Dev-02 投标：1.46
   Dev-03 投标：1.44
🏆 第 3 轮：中标

✅ 任务分配成功!
```

### 投标计算

```python
def calculate_bid(self, task: Dict) -> float:
    base_cost = task.get('estimated_time', 60) / 60.0
    skill_match = sum(self.capabilities.get(skill, 0.0) 
                      for skill in required_skills) / len(required_skills)
    workload_factor = 1.0 + len(self.current_tasks) * 0.2
    priority_factor = 1.0 - (priority - 5) * 0.05
    return base_cost * (1.0 - skill_match * 0.3) * workload_factor * priority_factor
```
