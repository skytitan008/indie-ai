## 📝 代码片段

### 自主学习

```python
class AutonomousLearner:
    def learn_topic(self, topic: str, max_pages: int = 5):
        # 搜索
        urls = self.crawler.search_duckduckgo(topic)
        
        # 抓取
        for url in urls:
            html = self.crawler.fetch_page(url)
            content = self.crawler.extract_content(html)
            
            # 存储
            self.knowledge_base.store_knowledge(
                topic=topic,
                content=content
            )
```

### 技能安装

```python
def install_skill(self, skill_name: str, functionality: str):
    # 生成代码
    code = self.skill_downloader.create_skill_template(
        skill_name, 
        functionality
    )
    
    # 保存
    path = self.skill_downloader.download_skill(
        skill_name, 
        code
    )
    
    # 注册
    self.knowledge_base.register_skill(
        name=skill_name,
        module_path=path
    )
```
