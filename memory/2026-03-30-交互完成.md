## 💡 代码片段

### 对话管理

```python
class ConversationManager:
    def add_message(self, user_input: str, ai_response: str):
        conn = sqlite3.connect(self.db_path)
        c = conn.cursor()
        c.execute('''
            INSERT INTO conversations (user_input, ai_response, topic)
            VALUES (?, ?, ?)
        ''', (user_input, ai_response, "chat"))
        conn.commit()
        conn.close()
```

### 聊天回复

def chat(self, text: str) -> str:
    if '你好' in text:
        return "你好！我是小七！"
    elif '谢谢' in text:
        return "不客气！😊"
    elif '你是谁' in text:
        return "我是小七，自主进化 AI！"
    else:
        return "我理解了，请继续说。"
