### 1. 任务规划核心模块

**文件:** `src/planning/task_planner.py` (18.2KB)

**核心类:**

#### TaskStatus - 任务状态
```python
PENDING      # 待执行
RUNNING      # 执行中
COMPLETED    # 已完成
FAILED       # 失败
BLOCKED      # 被阻塞
```

#### TaskPriority - 任务优先级
```python
CRITICAL = 1  # 关键
HIGH = 2      # 高
MEDIUM = 3    # 中
LOW = 4       # 低
```

#### Task - 任务类
- 任务 ID、名称、描述
- 状态、优先级
- 父子任务关系
- 依赖关系
- 时间估算
- 执行结果

#### TaskPlanner - 任务规划器
- `create_task()` - 创建任务
- `get_task()` - 获取任务
- `update_task_status()` - 更新状态
- `add_subtask()` - 添加子任务
- `get_ready_tasks()` - 获取就绪任务
- `get_all_tasks()` - 获取所有任务

#### TaskDecomposer - 任务分解器
- `decompose()` - 分解任务
- 8 种任务模板
- 智能关键词匹配

#### TaskExecutor - 任务执行器
- `execute_next()` - 执行下一个任务
- `complete_task()` - 完成任务
- `fail_task()` - 标记失败

### 方式 1: Python API

```python

    "开发用户登录功能",
    "开发一个完整的用户登录系统",
    "high"
)

# 执行任务

    pass

# 查看状态
status = ai.get_task_status()
```
