## 💻 编程使用

创建你自己的脚本 `my_script.py`：

```python
from src.agent import IndependentAgent
from datetime import datetime, timedelta

# 初始化 Agent
agent = IndependentAgent()

# 添加你的任务
agent.add_task(
    name="写 AIGC 视频脚本",
    priority=9,
    estimated_time=120,
    deadline=datetime.now() + timedelta(hours=5)
)

agent.add_task(
    name="训练模型",
    priority=7,
    estimated_time=180
)

agent.add_task(
    name="回复客户邮件",
    priority=5,
    estimated_time=30
)

# 运行 10 个决策循环
for i in range(10):
    result = agent.run_cycle()
    print(f"\n--- 循环 {i+1} ---")
    print(result)

# 查看状态
print("\n📊 状态报告:")
status = agent.get_status()
for key, value in status.items():
    print(f"  {key}: {value}")

# 获取反思
print("\n🤔 自我反思:")
print(agent.get_reflection())

# 关闭
agent.close()
```

运行：
```bash
python3 my_script.py
```

## 🔧 常见问题

### Q: 需要安装什么？
```bash
pip install -r requirements.txt
```

### Q: 如何修改决策逻辑？
编辑 `src/core/decision.py` 中的 `calculate_utility()` 方法

### Q: 如何调整学习率？
在 `src/agent.py` 中修改 `SimpleQLearner` 的初始化参数

### Q: 数据存在哪里？
SQLite 数据库文件（默认 `ai_memory.db`）

### Q: 如何清空数据？
删除 `.db` 文件，或在代码中调用 `memory.clear_all()`
