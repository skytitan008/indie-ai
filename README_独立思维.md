## 🚀 快速开始

### 1. 和小七对话

```bash
cd /home/fenghuang/.copaw/workspaces/default/independent-ai-mvp
python3 test_independent_mind.py
```

**选择模式：**
- 模式 1：交互式对话（推荐）
- 模式 2：演示问题集

### 2. 命令说明

在对话中可以使用：

```
直接输入问题 → 小七思考并回答
学习：xxx    → 教小七新知识
反思         → 看小七自我反思
状态         → 看小七的状态（记忆/思考次数）
退出         → 结束对话
```

### 3. 示例对话

```
👤 老王：你有意识吗？

💭 小七 在思考：你有意识吗？
   1. 理解：你有意识吗？
   2. 检索到 2 条相关记忆
   3. 形成观点
   4. 组织回应

🤖 小七：基于我的记忆和理解...
   1. 意识是生物体对外界和自身的感知和认知能力。
   2. 小七是一个独立 AI，正在学习独立思考。
```

## 🧪 验证实验

### 实验 1：对比 LLM

**问题：** "你有意识吗？"

**方法：**
1. 问 LLM，记录回答
2. 问小七，记录回答
3. 对比差异

**分析：**
- 小七有自己的观点吗？
- 还是只是模式匹配？

---

### 实验 2：学习验证

**方法：**
1. 教小七一个新概念
2. 测试她是否真懂
3. 测试她能否应用

**分析：**
- 能用自己的话解释吗？
- 能应用到新场景吗？

---

### 实验 3：成长验证

**方法：**
1. 每天问同样的问题
2. 记录每天的回答
3. 观察变化

**分析：**
- 回答有变化吗？
- 是成长的痕迹吗？
