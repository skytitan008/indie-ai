```javascript
const ws = new WebSocket('ws://localhost:8765/api/ws')

ws.onmessage = (event) => {
  const data = JSON.parse(event.data)
  
  if (data.type === 'emotion_update') {
    console.log('💝 小七情感:', data.emotion)
    console.log('🎭 Live2D 参数:', data.emotion.live2d_params)
    
    // 更新 Live2D 模型
    updateLive2D(data.emotion.live2d_params)
  }
}
```

## ️ 故障排除

### 问题 1: 端口被占用

```bash
# 查看占用端口的进程
lsof -ti:8765

# 杀死进程
lsof -ti:8765 | xargs kill -9

# 重新启动
./start_server.sh
```

---

### 问题 2: WebSocket 连接失败

检查防火墙：

```bash
# 允许 8765 端口
sudo ufw allow 8765
```

---

### 问题 3: Live2D 参数不生效

确保 AIRI 的 Live2D 模型支持这些参数。不同模型可能有不同的参数名。

常见参数名：

```
ParamEyeBrowForm
ParamEyeLOpen
ParamEyeROpen
ParamMouthForm
ParamCheek
ParamBodyAngleX
ParamArmLA
ParamArmRA
```
