## 2. 代码格式化

### 命令行使用

```bash
# 格式化单个文件
python3 -m src.integration.formatter src/agent.py

# 格式化整个目录
python3 -m src.integration.formatter src/ demo/

# 只检查不修改
python3 -m src.integration.formatter src/ --check
```

### Python API

```python
from src.integration.formatter import CodeFormatter

# 创建格式化器
formatter = CodeFormatter({
    'python_formatter': 'black',  # 或 autopep8
    'js_formatter': 'prettier'
})

# 格式化单个文件
formatter.format_python('src/agent.py')

# 格式化目录
stats = formatter.format_directory('src/', extensions=['.py'])

# 查看报告
print(formatter.get_report())
```

### 配置选项

```python
config = {
    'python_formatter': 'black',      # Python 格式化器
    'js_formatter': 'prettier',       # JS 格式化器
    'line_length': 88,                # 行宽
    'skip_string_normalization': True # 跳过字符串规范化
}
```

### 快捷函数

```python
from src.integration.formatter import format_code

# 快速格式化
formatter = format_code(['src/', 'demo/'])
print(formatter.get_report())
```

## 3. 测试运行

### 命令行使用

```bash
# 运行所有测试
python3 -m src.integration.test_runner

# 运行指定目录
python3 -m src.integration.test_runner tests/

# 生成覆盖率报告
python3 -m src.integration.test_runner tests/ --coverage
```

### Python API

```python
from src.integration.test_runner import TestRunner

# 创建测试运行器
runner = TestRunner({
    'framework': 'pytest',    # 或 unittest
    'test_dir': 'tests',
    'coverage': True,
    'verbose': True
})

# 运行测试
results = runner.run_tests()

# 查看报告
print(runner.get_report())

# 保存 JSON 报告
runner.save_report('test_report.json')
```

### 配置选项

```python
config = {
    'framework': 'pytest',        # 测试框架
    'test_dir': 'tests',          # 测试目录
    'coverage': True,             # 是否生成覆盖率
    'verbose': True,              # 详细输出
    'timeout': 300                # 超时时间（秒）
}
```

### 快捷函数

```python
from src.integration.test_runner import run_tests

# 快速运行测试
runner = run_tests('tests/', framework='pytest')
print(runner.get_report())
```

## 4. 日报生成

### 命令行使用

```bash
# 生成今日日报
python3 -m src.integration.daily_report

# 生成指定日期日报
python3 -m src.integration.daily_report --date 2026-03-29
```

### Python API

```python
from src.integration.daily_report import DailyReportGenerator
from datetime import datetime

# 创建生成器
generator = DailyReportGenerator({
    'project_root': '.',
    'git_repo': True,
    'include_experiments': True,
    'include_files': True
})

# 生成日报
report = generator.generate()
print(report)

# 保存到文件
output_file = generator.save_report()
print(f"日报已保存到：{output_file}")

# 生成指定日期
date = datetime(2026, 3, 29)
report = generator.generate(date)
```

### 日报内容

生成的日报包含：
- 📊 今日概览（提交数、实验数、文件修改）
- 💻 Git 提交记录
- 🧪 实验运行记录
- 📁 文件修改记录
- 📋 明日计划
- 💭 今日总结

### 自定义模板

```python
class CustomReportGenerator(DailyReportGenerator):
    def _generate_tomorrow_plan(self):
        # 自定义明日计划部分
        lines = []
        lines.append("━━━ 📋 明日计划 ━━━")
        lines.append("")
        lines.append("  [ ] 完成 Windows 桌面版")
        lines.append("  [ ] 优化学习算法")
        lines.append("  [ ] 写技术博客")
        return '\n'.join(lines)
```

## 5. Git Hooks

### 安装

```bash
# 安装 Git Hooks
python3 -m src.integration.git_hooks install

# 查看状态
python3 -m src.integration.git_hooks status

# 卸载
python3 -m src.integration.git_hooks uninstall
```

### Python API

```python
from src.integration.git_hooks import GitHooks

# 创建管理器
hooks = GitHooks()

# 安装钩子
hooks.install(['pre-commit', 'post-commit', 'pre-push'])

# 查看状态
hooks.status()

# 卸载
hooks.uninstall()
```

### 钩子功能

#### pre-commit（提交前）
- ✅ 检查 Python 代码格式（black）
- ✅ 运行相关测试（pytest）
- ✅ 检查 JS/TS 代码格式（prettier）

#### post-commit（提交后）
- ✅ 记录提交信息到日志
- ✅ 保存为 JSON 格式

#### pre-push（推送前）
- ✅ 运行完整测试套件
- ✅ 检查敏感信息（密码、密钥等）
- ✅ 确认后才允许推送

### 自定义钩子

在 `.githooks/` 目录下创建脚本：

```bash
# .githooks/pre-commit
#!/bin/bash
echo "运行自定义检查..."
# 你的检查逻辑
```

## 6. 任务调度

### 启动调度器

```bash
# 持续运行（守护进程）
python3 -m src.integration.task_scheduler

# 运行一次（测试）
python3 -m src.integration.task_scheduler --once
```

### Python API

```python
from src.integration.task_scheduler import TaskScheduler

# 创建调度器
scheduler = TaskScheduler()

# 安排晨间任务（14:00 - 老王的工作时间）
scheduler.schedule_morning_routine("14:00")

# 安排晚间任务（02:00）
scheduler.schedule_evening_routine("02:00")

# 添加自定义任务
def my_custom_task():
    print("执行自定义任务...")

scheduler.add_task(
    name='自定义任务',
    callback=my_custom_task,
    interval='daily',
    time_str='15:00'
)

# 启动调度器
scheduler.start()
```

### 调度任务

```python
# 每小时运行
scheduler.add_task('每小时任务', callback, interval='hourly')

# 每天运行
scheduler.add_task('每天任务', callback, interval='daily', time_str='10:00')

# 每周运行
scheduler.add_task('每周任务', callback, interval='weekly')
```

### 晨间任务

自动执行：
- 📋 检查待办任务
- 📅 规划今日任务
- 🔔 发送晨间提醒

### 晚间任务

自动执行：
- 📝 生成日报
- 🧹 清理临时文件
- 🔔 发送晚间提醒

python3 -m src.integration.formatter src/ demo/

python3 -m src.integration.test_runner tests/

python3 -m src.integration.daily_report

### 团队协作

#### 统一配置

创建 `pyproject.toml`:
```toml
[tool.black]
line-length = 88
target-version = ['py312']

[tool.pytest.ini_options]
testpaths = ["tests"]

```

#### CI/CD 集成

`.github/workflows/ci.yml`:
```yaml
name: CI

on: [push, pull_request]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      
      - name: Set up Python
        uses: actions/setup-python@v2
        with:
          python-version: 3.12
      
      - name: Install dependencies
        run: pip install -r requirements.txt
      
      - name: Format check
        run: black --check src/ demo/
      
      - name: Run tests
        run: pytest tests/ -v
```

## 🐛 常见问题

### Q1: black 格式化失败？

**A**: 确保已安装 black：
```bash
pip install black
```

### Q2: Git Hooks 不工作？

**A**: 检查配置：
```bash
git config core.hooksPath
# 应该显示：.githooks
```

### Q3: 日报生成失败？

**A**: 确保在项目根目录运行：
```bash
cd /path/to/indie-ai
python3 -m src.integration.daily_report
```

### Q4: 测试运行超时？

**A**: 增加超时时间：
```python
runner = TestRunner({'timeout': 600})  # 10 分钟
```
