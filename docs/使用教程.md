## 2. 核心概念

### 2.1 IndependentAgent

**核心 AI 类**，整合所有功能模块：

```python
from src.agent import IndependentAgent

# 创建 Agent
agent = IndependentAgent()

# 查看状态
print(f"状态：{agent.state}")
print(f"经验：{agent.experience}")
```

### 2.2 决策引擎

**透明决策机制**，基于效用函数：

```python
from src.decision import DecisionEngine

engine = DecisionEngine()

# 设置权重（质量优先）
engine.weights = {
    'urgency': 0.30,      # 紧急度 30%
    'importance': 0.40,   # 重要性 40%
    'efficiency': 0.20,   # 效率 20%
    'dependency': 0.10,   # 依赖性 10%
}

# 计算效用值
utility = engine.calculate_utility(task)
```

### 2.3 学习系统

**Q-Learning + SARSA** 双算法：

```python
from src.learning.ql earner import SimpleQLearner
from src.learning.sarsa import SARSALearner

# Q-Learning (Off-policy)
q_learner = SimpleQLearner()
q_learner.update(state, action, reward, next_state)

# SARSA (On-policy)
sarsa_learner = SARSALearner()
sarsa_learner.update(state, action, reward, next_state, next_action)
```

### 2.4 多 Agent 协作

**团队协作系统**：

```python
from src.multi_agent.coordinator import MultiAgentCoordinator, CollaborativeAgent, AgentRole

# 创建协调器
coordinator = MultiAgentCoordinator()

# 注册 Agent
tech_agent = CollaborativeAgent(
    agent_id="TechAgent-01",
    role=AgentRole.SPECIALIST,
    skills=["python", "debugging", "pipeline"]
)

coordinator.register_agent(tech_agent)

# 分配任务
task = Task(description="修复 ComfyUI bug")
assigned = coordinator.assign_task(task)
```

## 3. 使用指南

### 3.1 配置系统

**5 种预设配置**：

```python
from src.config import get_preset, AgentConfig

# 方式一：使用预设
config = get_preset('quality')  # 质量优先（推荐）

# 方式二：自定义
config = AgentConfig(
    learning_rate=0.1,
    discount_factor=0.95,
    exploration_rate=0.3,
    decision={
        'urgency_weight': 0.30,
        'importance_weight': 0.40,
        'efficiency_weight': 0.20,
        'dependency_weight': 0.10,
    }
)
```

**配置对比**:

| 配置名 | 紧急度 | 重要性 | 效率 | 适用场景 |
|--------|--------|--------|------|----------|
| `balanced` | 35% | 35% | 20% | 通用 |
| `urgent` | 50% | 25% | 15% | 紧急任务 |
| `quality` | 30% | 40% | 20% | **创意工作** ⭐ |
| `aggressive` | 25% | 30% | 35% | 快速执行 |
| `conservative` | 30% | 35% | 15% | 稳健执行 |

### 3.2 任务管理

**创建任务**:

## 5. 常见问题

### Q1: 安装失败怎么办？

**A**: 检查 Python 版本：
```bash
python3 --version  # 需要 3.12+
```

升级 Python 或创建虚拟环境：
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### Q2: 学习曲线不收敛？

**A**: 调整学习率：
```python
config = get_preset('quality')
config.learning.learning_rate = 0.05  # 降低学习率
```

### Q3: 多 Agent 协作不工作？

**A**: 检查 Agent 技能匹配：
```python
# 确保任务技能在 Agent 技能列表中
agent.skills = ["python", "debugging"]
task.required_skills = ["python"]  # ✓ 匹配
```

### Q4: Web 界面打不开？

**A**: 检查端口：
```bash
# 端口 8000 可能被占用
python3 start_web.py --port 8080  # 改用 8080
```

### Q5: 如何保存学习成果？

**A**: 使用 SQLite 持久化：
```python
from src.memory.database import MemoryDatabase

db = MemoryDatabase("my_agent.db")
db.save_q_table(agent.q_table)
db.save_experience(agent.experience)
```

## 6. 最佳实践

### 6.1 配置选择

**AIGC 创意工作** → 使用 `quality` 配置
```python
config = get_preset('quality')
```

**紧急任务处理** → 使用 `urgent` 配置
```python
config = get_preset('urgent')
```

**日常开发** → 使用 `balanced` 配置
```python
config = get_preset('balanced')
```

### 6.2 学习参数调优

**初期探索**（前 20 轮）:
```python
exploration_rate = 0.5  # 高探索
learning_rate = 0.2     # 快学习
```

**中期收敛**（20-60 轮）:
```python
exploration_rate = 0.2  # 中探索
learning_rate = 0.1     # 中学习
```

**后期稳定**（60+ 轮）:
```python
exploration_rate = 0.05  # 低探索
learning_rate = 0.05     # 慢学习
```

### 6.3 任务优先级设置

**P10** - 紧急且重要（今天必须完成）
**P9-P7** - 重要任务（本周完成）
**P6-P4** - 常规任务（本月完成）
**P3-P1** - 低优先级（有空再做）

### 6.4 多 Agent 协作

**技能互补**:
```python
# 技术专家
tech_agent.skills = ["python", "debugging", "pipeline"]

# 创意专家
creative_agent.skills = ["design", "script", "prompt"]
```

**负载均衡**:
```python
# 监控 Agent 负载
for agent in coordinator.agents:
    print(f"{agent.id}: {len(agent.assigned_tasks)} 任务")
```
