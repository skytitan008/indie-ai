### 集成方式

**方案 1: API 服务（已实现）**

```python
# 小七独立运行 API 服务
uvicorn indie_ai_server:app --host 0.0.0.0 --port 8765

# AIRI 通过 API 调用小七
POST /api/chat → 小七思考 → 返回回答 + 情感状态
GET /api/status → 获取小七状态 → 更新 Live2D 表情
WebSocket /api/ws → 实时推送思考过程
```

**方案 2: 插件式**

```python
# 小七作为 AIRI 插件
airi/plugins/indie_ai_core.py

class IndieAICore:
    def on_message(self, message: str):
        # 用小七的自主思维回应
        return self.mind.respond(message)
```
