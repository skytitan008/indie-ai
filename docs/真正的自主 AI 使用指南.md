## 📊 自主行为类型

### 1. 执行任务
```
条件：有就绪任务
行动：执行下一个任务
输出：✅ 执行任务：XXX
```

### 2. 学习
```
条件：精力充足，需要知识
行动：学习新主题
输出：📚 学习：XXX
```

### 3. 自我改进
```
条件：每 10 次行动
行动：自我改进
输出：🔄 自我改进
```

### 4. 创建任务
```
条件：没有任务
行动：创建新任务
输出：📋 创建任务：XXX
```

### 5. 探索
```
条件：20% 概率
行动：探索新领域
输出：🔍 探索：XXX
```

### 6. 休息
```
条件：精力不足
行动：休息恢复
输出：💤 休息中...
```
