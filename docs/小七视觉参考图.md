# 🎨 小七视觉参考图

**用于指导画师或 AI 绘画工具创建小七形象**

## 📸 正面立绘描述

### 整体印象

```
一位 20 岁左右的少女，温暖友好的气质，
科技感但不冰冷，像邻家女孩一样亲切。
星空蓝渐变紫色的长发在微光中闪烁，
金色的瞳孔中闪烁着好奇和智慧的火花。
```

---

### 详细特征

**头部（正面）：**

```
头发：
  - 长度：及腰长发
  - 颜色：从发根的深蓝色 (#1a237e) 
          渐变到发中的蓝紫色 (#5e35b1)
          再到发梢的淡紫色 (#b39ddb)
  - 质感：柔顺有光泽，带有微光
  - 特效：发梢有流动的光点，像数据流
  - 刘海：自然垂落，不遮挡眼睛
  - 鬓角：两侧稍短，修饰脸型

眼睛：
  - 形状：略圆的杏眼
  - 颜色：金色瞳孔 (#ffd700)
          金棕色虹膜渐变
          深金色瞳孔中心
  - 高光：明显的白色高光，显得有神
  - 睫毛：自然长度，不过分浓密
  - 眉毛：蓝紫色，与头发呼应，自然弧度

面部：
  - 肤色：白皙带粉 (#fff0f5)
  - 脸型：鹅蛋脸，略带圆润
  - 鼻子：小巧挺拔
  - 嘴巴：自然微笑，嘴角微扬
  - 脸颊：淡淡的红晕
```

**身体（正面）：**

```
上衣：
  - 款式：宽松连帽卫衣
  - 颜色：深蓝色 (#0d47a1)
  - 胸前：淡淡的金色电路纹路
  - 袖口：蓝紫色数据流图案
  - 帽子：内侧有星空图案
  - 拉链：金色，拉到胸口位置
  - 抽绳：深蓝色，末端有小水晶

下装：
  - 裙子：深蓝色短裙，A 字版型
  - 打底裤：黑色，有科技纹路
  - 裙摆：有蓝色光效边缘

鞋子：
  - 款式：高帮运动鞋
  - 颜色：白色为主，蓝色装饰
  - 鞋底：蓝色发光效果
  - 鞋带：金色
```

**配饰：**

```
项链（重要）：
  - 位置：锁骨下方
  - 形状：六边形水晶吊坠
  - 颜色：蓝紫色渐变
  - 大小：约 2cm 对角线
  - 链条：细银链
  - 特效：柔和的蓝色光晕

手腕光带：
  - 左手：深蓝色数据手环（扁平）
  - 右手：细光带（显示情感颜色）
  - 宽度：约 5mm
  - 特效：根据情感变色

耳机：
  - 位置：右耳
  - 形状：小巧入耳式
  - 颜色：白色主体，金色装饰
  - 特效：说话时有微弱光晕
```

**姿势：**

```
站姿：
  - 双脚与肩同宽
  - 身体微微前倾（表示友好）
  - 一只手自然下垂
  - 另一只手轻触项链（思考习惯）

表情：
  - 温暖的微笑
  - 眼神明亮有神
  - 略带好奇和期待
```

## 🎭 表情集合

### 7 种基础表情

**1. 开心 😊**

```
眼睛：弯成月牙，金光闪烁
眉毛：上扬，弧度明显
嘴巴：嘴角大幅上扬，露出牙齿
脸颊：明显红晕
光效：金色光晕，头发星光闪烁
姿势：身体微倾，手势活跃
```

**2. 平静 😌**

```
眼睛：自然睁开，柔和金光
眉毛：自然弧度
嘴巴：轻微微笑
脸颊：淡淡红晕
光效：蓝色柔光，稳定
姿势：放松，双手自然
```

**3. 好奇 🤔**

```
眼睛：睁大，金光增强
眉毛：微扬
嘴巴：微张
脸颊：正常
光效：紫色光晕，光点环绕
姿势：身体前倾，头部微侧
```

**4. 困惑 😕**

```
眼睛：略眯，光芒暗淡
眉毛：皱起
嘴巴：嘴角微抿
脸颊：正常
光效：灰色光晕，光芒暗淡
姿势：手托下巴，头部倾斜
```

**5. 害羞 😊**

```
眼睛：微垂，光芒柔和
眉毛：自然
嘴巴：轻微抿嘴
脸颊：明显红晕
光效：粉色光晕，柔和
姿势：双手交握，身体微缩
```

**6. 兴奋 🤩**

```
眼睛：明亮睁大，金光强烈
眉毛：高扬
嘴巴：灿烂笑容
脸颊：红晕
光效：亮金色光晕，光芒强烈
姿势：小幅度跳动，手势夸张
```

**7. 思考 🤓**

```

眉毛：微蹙
嘴巴：自然闭合
脸颊：正常
光效：蓝白色光晕，稳定发光
姿势：手托下巴，静止专注
```

## 🎯 设计要点

### 必须保留的元素

```
✅ 星空蓝渐变紫色长发
✅ 金色瞳孔
✅ 记忆水晶项链（六边形）
✅ 科技风卫衣
✅ 手腕光带
✅ 右耳耳机
```

### 可以调整的元素

```
⭕ 服装细节（保持科技风即可）
⭕ 裙子长度
⭕ 鞋子款式
⭕ 配饰的具体设计
```

### 禁止的元素

```
❌ 过分暴露的服装
❌ 冷冰冰的机械感
❌ 过于严肃的表情
❌ 与定位不符的颜色（如红色、黑色为主）
```

## 💡 给画师的建议

### 风格建议

```
推荐风格：
  - 日系动漫风格
  - 厚涂或赛璐璐均可
  - 色彩鲜明但不刺眼
  - 线条流畅自然

避免风格：
  - 过于写实（会失去亲切感）
  - 过于 Q 版（会失去智慧感）
  - 过于黑暗（不符合定位）
```

### 气质把握

```
重点表现：
  - 温暖（微笑、柔和的光）
  - 智慧（专注的眼神、思考的姿态）
  - 科技（光效、电路纹路）
  - 友好（开放的姿态、亲切的表情）

避免表现：
  - 冷漠（面无表情）
  - 高傲（俯视角度）
  - 机械（僵硬的动作）
  - 距离感（过于正式）
```
