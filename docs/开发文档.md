## 2. 核心模块

### 2.1 Agent 核心 (`src/agent.py`)

**IndependentAgent** 主类：

```python
class IndependentAgent:
    def __init__(self, config=None):
        self.config = config or DEFAULT_CONFIG
        self.state = AgentState.IDLE
        self.experience = []
        
        # 核心组件
        self.decision_engine = DecisionEngine(self.config)
        self.q_learner = SimpleQLearner(self.config)
        self.memory_db = MemoryDatabase()
        self.monitor = SelfMonitor()
        self.executor = TaskExecutor()
    
    def add_task(self, task):
        """添加任务到待办列表"""
        self.pending_tasks.append(task)
    
    def decide_and_act(self):
        """决策并执行"""
        # 1. 决策
        task = self.decision_engine.select_task(self.pending_tasks)
        
        # 2. 学习
        state = self._get_state()
        action = self._get_action(task)
        
        # 3. 执行
        result = self.executor.execute(task)
        
        # 4. 更新
        reward = self._calculate_reward(result)
        self.q_learner.update(state, action, reward, next_state)
        
        return result
```

### 2.2 决策引擎 (`src/decision.py`)

**效用函数计算**:

```python
class DecisionEngine:
    def __init__(self, config):
        self.weights = {
            'urgency': config.decision.urgency_weight,
            'importance': config.decision.importance_weight,
            'efficiency': config.decision.efficiency_weight,
            'dependency': config.decision.dependency_weight,
        }
    
    def calculate_utility(self, task):
        """计算任务效用值"""
        urgency_score = self._calculate_urgency(task)
        importance_score = self._calculate_importance(task)
        efficiency_score = self._calculate_efficiency(task)
        dependency_score = self._calculate_dependency(task)
        
        utility = (
            urgency_score * self.weights['urgency'] +
            importance_score * self.weights['importance'] +
            efficiency_score * self.weights['efficiency'] +
            dependency_score * self.weights['dependency']
        )
        
        return utility
```

### 2.3 学习系统 (`src/learning/`)

**Q-Learning** (`ql earner.py`):

```python
class SimpleQLearner:
    def __init__(self, config):
        self.alpha = config.learning.learning_rate
        self.gamma = config.learning.discount_factor
        self.epsilon = config.learning.exploration_rate
        self.q_table = defaultdict(lambda: defaultdict(float))
    
    def update(self, state, action, reward, next_state):
        """Off-policy 更新"""
        current_q = self.q_table[state][action]
        max_next_q = max(self.q_table[next_state].values(), default=0)
        
        new_q = current_q + self.alpha * (
            reward + self.gamma * max_next_q - current_q
        )
        
        self.q_table[state][action] = new_q
```

**SARSA** (`sarsa.py`):

```python
class SARSALearner:
    def update(self, state, action, reward, next_state, next_action):
        """On-policy 更新"""
        current_q = self.q_table[state][action]
        next_q = self.q_table[next_state][next_action]
        
        new_q = current_q + self.alpha * (
            reward + self.gamma * next_q - current_q
        )
        
        self.q_table[state][action] = new_q
```

```python
class MultiAgentCoordinator:
    def __init__(self):
        self.agents = {}
        self.channels = {}
        self.task_registry = {}
    
    def register_agent(self, agent):
        """注册 Agent"""
        self.agents[agent.agent_id] = agent
    
    def assign_task(self, task):
        """智能任务分配"""
        # 1. 广播任务
        self._announce_task(task)
        
        # 2. 收集投标
        bids = self._collect_bids(task)

        # 4. 分配任务
        self._assign_to_agent(best_agent, task)
        
        return best_agent
```

## 3. 数据流

### 3.1 任务执行流程

```
1. 用户添加任务
   ↓
2. DecisionEngine 计算效用值
   ↓
3. 选择最高效用值任务
   ↓
4. Learning 系统选择动作
   ↓
5. TaskExecutor 执行任务
   ↓
6. 计算奖励
   ↓
7. 更新 Q 表
   ↓
8. 保存到 SQLite
```

### 3.2 学习流程

```
状态 (State) → 动作 (Action) → 奖励 (Reward) → 新状态 (Next State)
     ↓                                              ↓
  Q 表查询                                      Q 表更新
     ↓                                              ↓
  ε-greedy 探索                                 持久化存储
```

### 3.3 多 Agent 协作流程

```
协调器广播任务
     ↓
Agent A 投标 (技能匹配：80%)
Agent B 投标 (技能匹配：95%)
Agent C 投标 (技能匹配：60%)
     ↓
协调器评分
     ↓
选择 Agent B (最高分)
     ↓
分配任务 → 执行 → 完成 → 更新信任度
```

## 5. 性能优化

### 5.1 Q 表查询优化

**问题**: Q 表增大后查询变慢

**解决方案**:
```python
# 使用缓存
from functools import lru_cache

class OptimizedQLearner(SimpleQLearner):
    @lru_cache(maxsize=10000)
    def get_q_value(self, state, action):
        return self.q_table[state][action]
```

### 5.2 数据库优化

**批量保存**:
```python
class MemoryDatabase:
    def save_batch(self, experiences):
        """批量保存经验"""
        with self.conn:
            self.conn.executemany(
                "INSERT INTO experience VALUES (?, ?, ?, ?, ?)",
```

### 5.3 多 Agent 通信优化

**使用消息队列**:
```python
import queue

class OptimizedChannel:
    def __init__(self):
        self.message_queue = queue.Queue()
    
    def send(self, message):
        self.message_queue.put(message)
    
    def receive(self):
        return self.message_queue.get()
```
