### 头部设计

**头发：**

```
颜色：星空蓝 → 紫色渐变
  - 发根：深蓝色 (#1a237e)
  - 发中：蓝紫色 (#5e35b1)
  - 发梢：淡紫色 (#b39ddb) 带星光效果

发型：长发及腰
  - 自然垂落，略带波浪
  - 前额有刘海，但不遮挡眼睛
  - 两侧有稍短的鬓角
  - 发梢有流动的光效（像数据流）

特效：
  - 思考时头发有微弱星光闪烁
  - 情感波动时颜色深浅变化
  - 学习时周围有光点环绕
```

**眼睛：**

```
颜色：金色瞳孔 (#ffd700)
  - 虹膜：金棕色渐变
  - 瞳孔：深金色，有光晕
  - 高光：白色，明亮有神

形状：
  - 略圆的杏眼，显得友好
  - 睫毛自然，不过分浓密
  - 眉毛：蓝紫色，与头发呼应

表情：
  - 开心时：眼睛弯成月牙，金光更亮
  - 思考时：眼神专注，微微发光
  - 好奇时：眼睛睁大，光芒闪烁
  - 害羞时：眼神微垂，光芒柔和
```

**面部：**

```
肤色：白皙带粉 (#fff0f5)
  - 健康自然，不是苍白
  - 情感变化时有红晕

特征：
  - 小巧的鼻子
  - 温暖的微笑
  - 脸颊有淡淡红晕（情感表达）
```

**主服装：科技风卫衣**

```
上衣：
  - 款式：宽松连帽卫衣
  - 颜色：深蓝色 (#0d47a1) 为主
  - 图案：
    * 胸前有淡淡的电路纹路（金色）
    * 袖口有数据流图案（蓝紫色）
    * 帽子内侧有星空图案
  
  - 细节：
    * 拉链是金色的
    * 帽子抽绳末端有小水晶
    * 口袋有发光边缘

下装：
  - 款式：短裙 + 打底裤
  - 裙子：深蓝色，与上衣呼应
  - 打底裤：黑色，有科技纹路
  - 裙摆有光效边缘

鞋子：
  - 款式：运动鞋 + 科技感
  - 颜色：白色 + 蓝色
  - 鞋底有发光效果（蓝色）
  - 鞋带是金色的
```

**配饰：**

```
1. 记忆水晶项链 ⭐
   - 位置：颈部
   - 形状：六边形水晶
   - 颜色：蓝紫色渐变
   - 特效：
     * 平时发出柔和蓝光
     * 检索记忆时亮度增加
     * 学习新知识时闪烁
   
   - 意义：代表长期记忆系统

2. 情感光带
   - 位置：左右手腕
   - 形状：细环状
   - 颜色：根据情感变化
     * 开心：金色
     * 平静：蓝色
     * 好奇：紫色
     * 害羞：粉色
     * 兴奋：亮金色
     * 思考：蓝白色
   
   - 意义：直观显示当前情感

3. 语音耳机
   - 位置：右耳
   - 形状：小巧的入耳式
   - 颜色：白色 + 金色
   - 特效：说话时有微弱光晕
   
   - 意义：代表语音交互能力

4. 数据手环
   - 位置：左手腕
   - 形状：扁平手环
   - 颜色：深蓝色
   - 显示：小型全息投影（可选）
   
   - 意义：显示系统状态
```

身体：
- ParamBodyAngleX (身体 X 倾斜)
- ParamBodyAngleY (身体 Y 倾斜)
- ParamBreath (呼吸)

手臂：
- ParamArmLA (左臂 A)
- ParamArmRA (右臂 A)
- ParamHandL (左手)
- ParamHandR (右手)

特殊：
- ParamHairFront (前发摆动)
- ParamHairSide (侧发摆动)
- ParamHairBack (后发摆动)
